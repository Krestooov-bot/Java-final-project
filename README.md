# University Management System 🎓

Консольное приложение (CLI) на Java для имитации простой системы управления университетом. Проект позволяет работать со студентами, преподавателями, курсами и зачислениями, а также включает базовую систему отчетов.

## 🚀 Основной функционал

* **Студенты:** Добавление, удаление, просмотр, сортировка по ФИО (Bubble Sort) и поиск по имени/email.
* **Преподаватели:** Регистрация преподавателей с указанием их должности (Assistant, Lecturer, Professor).
* **Курсы:** Создание учебных курсов с привязкой к конкретному преподавателю и указанием количества кредитов.
* **Зачисления и Оценки:** Запись студентов на курсы, выставление оценок и отметка об оплате.
* **Отчеты:** Вывод подробного транскрипта студента с автоматическим расчетом среднего балла (GPA) и просмотр списка неоплаченных курсов.

## 🛠️ Технологии и подходы

* **Язык:** Java
* **Архитектура:** Принципы объектно-ориентированного программирования (ООП)
* **Особенности реализации:**
  * Использование `enum` для статусов, должностей и оценок.
  * Применение интерфейса `Payable` для финансовых операций.
  * Наследование (использование базового абстрактного класса `Person` для `Student` и `Teacher`).
  * Обработка исключительных ситуаций для защиты от неверного пользовательского ввода.

## ⚙️ Как запустить проект

1. Склонируйте этот репозиторий на свой компьютер.
2. Откройте проект в любой удобной Java IDE (рекомендуется **IntelliJ IDEA**, **Eclipse** или **VS Code**).
3. Перейдите к файлу `Main.java` в пакете `university`.
4. Запустите метод `main`.
5. Используйте цифры на клавиатуре для навигации по консольному меню.
